## Python

### if

```python
if 2 > 1:
    print("2 greater than 1")
```
```d
if (2 > 1) {
    writeln("2 greater than 1")
}
```

### foreach
```python
for item in some_list:
    print(item)
```
```d
foreach(item; someList) {
    writeln(item);
}
```

### foreach with indieces
```python
for i, item in enumerate(some_list):
    print(i, item)
```
```d
foreach(i, item; someList) {
    writeln(i, item);
}
```
### while
```python
while True:
    print(1)
```

```d
while(true) {
    writeln(1);
}
```
